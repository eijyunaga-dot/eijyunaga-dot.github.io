## 写真を撮影▶スタンプで名前付けしたりするWEBアプリです
 主に施設での荷物確認を想定しています
  (現在メモリ使用量の最適化を考案中) 

## iOSでのインストール方法

このアプリをデバイスで使用するには、以下の手順を行ってください。

1.  **Safariで開く**:
    iPhone/iPadのSafariで、アップロードしたURLにアクセスします。
    (PCの場合はアクセス後にそのまま使用開始できます)

3.  **ホーム画面に追加**:
    画面下部の「共有」ボタン（四角から矢印が出ているアイコン）をタップし、「ホーム画面に追加」を選択します。

4.  **アプリの起動**:
    ホーム画面に追加されたアイコンをタップすると、アドレスバーのないネイティブアプリのような全画面モードで起動します。

## 使い方

### スタンプの編集
1.  「写真スタンプ」を選択します。
2.  画像をアップロードします。
3.  **スタンプの追加**: 「+ スタンプ追加」ボタンを押すと、スタンプを追加できます（最大8つまで）。
4.  **スタンプの選択**: 編集したいスタンプをタップするか、ドラッグして選択します。選択中のスタンプは点線で囲まれます。
5.  **スタンプの削除**: 複数ある場合、「削除」ボタンで選択中のスタンプを削除できます。
6.  「定型文」または「自由入力」で文字を入力します。よく使う定型文は下記の設定から追加できます。
7.  フォントサイズ、透明度、色を調整します。
8.  文字をドラッグして位置を調整します。
9.  スマホ用、PC用、古いiPad等用の保存をそれぞれタップして画像を保存します（iOSでは共有シートが開きます）。保存後、画面は自動的にリセットされます。
10.  最近のiPhoneの高解像度画像やHEIF形式の画像は、読込時にあらかじめリサイズされます(クラッシュ防止)

### 定型文の管理
1.  「設定」を選択します。
2.  **追加**: 入力欄に文字を入力し、「追加」ボタンを押します。
3.  **削除**: リストの「削除」ボタンを押すと、その定型文が削除されます。
4.  **初期化**: 「初期設定に戻す」ボタンで、デフォルトの定型文リストに戻せます。
5.  **言語変更**: 日本語以外に設定すると説明文が変わります（デフォルトスタンプ文字はそのまま）

### 画像補正
**画像補正**: 暗い画像や逆光の画像を補正します(細かい調整は各端末の写真アプリを使用して下さい)、大きい解像度画像やHEIFは変換・リサイズされます
